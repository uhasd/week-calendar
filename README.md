# Week Calendar

You can visit the calendar [here](https://uhasd.github.io/week-calendar/calendar.html)

## Note
- Data is stored in localStorage

- Made with Bootstrap v5 (CSS Only)

- Has Drag and Drop functionality\
    Note: Does not work on touch inputs\
    i.e. Mobile Devices

## Usage

The calendar has two pages the [calendar](https://uhasd.github.io/week-calendar/calendar.html) page and the [create event](https://uhasd.github.io/week-calendar/create-event.html) page.

**Calendar:** This is the calendar's main page, where all of the events are displayed and where they can be deleted.

**Create Event:** Just as the name suggests, it is where you can create an event for the calendar.
